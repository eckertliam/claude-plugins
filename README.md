# Claude Plugins

Personal Claude Code plugin marketplace.

## Plugins

- **devplan** - File-based development planning. Create, follow, and update persistent plan files in the working directory.
- **reflect** - End-of-session retrospective. Reviews what happened, interviews the user, then suggests improvements to skills, CLAUDE.md, and proposes new skills.

## Installation

```
/plugin install devplan@claude-plugins
```
